🚀 How to Run

🐳Docker (Recommended)
💻From Source

Run with Docker

# Pull the image
docker pull charlie2233/hoops-clips:latest

# Run the server
docker run -p 8787:8787 -v ./videos:/app/input charlie2233/hoops-clips

# Send a video for processing
curl -X POST http://localhost:8787/api/ai/analyze \
  -H "Content-Type: application/json" \
  -d '{"videoUrl":"https://example.com/game.mp4"}'

Requires Docker and ~8GB disk space for model weights.

Build from Source

# Clone the repository
git clone https://github.com/charlie2233/Basketball_action_recoginition_sever.git
cd Basketball_action_recoginition_sever

# Create virtual environment
python -m venv venv && source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Download model weights
python scripts/download_weights.py

# Run the server
uvicorn ai_server:app --host 0.0.0.0 --port 8787

Requires Python 3.9+, CUDA 11.8+ for GPU acceleration.
